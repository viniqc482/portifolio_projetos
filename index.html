Segmentação de Clientes para Redução de Churn e Aumento da Retenção

Análise descritiva no Power BI

Através de um dashboard, mostrei detalhes de locações de vários Airbnbs organizados por uma empresa de locações. A quantidade imensa de quartos, valores e regiões, os tornavam difíceis de analisar com eficiência. Com uma organização estratégica, distribui os dados em vários filtros, como data, ID, nome e tipo dos quartos, bairros e regiões, os quais interagiram com alguns gráficos, como o de colunas para uma análise de pareto, gráficos de disco para análise por regiões, entre outros. Assim, o time de vendas obteve maior facilidade na criação de novas estratégias.

Dashboard Logístico de Entregas com Python

Dashboard interativo desenvolvido em Python para análise de dados de entregas, utilizando um dataset em formato CSV. O projeto envolve limpeza, tratamento e modelagem dos dados, além da construção de métricas e visualizações para análise operacional e tomada de decisão. A aplicação foi construída com Streamlit e publicada em ambiente cloud para acesso público.
